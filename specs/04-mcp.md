# Slide 4: From Bundled Tools to Remote MCP

## Type
Evolution diagram

## Content
- **Title**: "From Bundled to Shared"
- Show the evolution:

  **Before MCP:**
  - Tools bundled inside each app
  - Every app reimplements the same integrations
  - Diagram: multiple apps each with their own tool implementations

  **After MCP (Remote):**
  - Tools hosted as remote servers
  - Any agent can connect to any MCP server
  - "Share tools with agents you've never met"
  - Diagram: multiple agents → single MCP server

- Key quote/callout: "Remote MCP lets people share tools with agents they've never met"

## Layout
- Two-panel before/after, or a left-to-right evolution arrow
- Use compute/ai color scheme from the design system

## Key point
- MCP standardized tool sharing
- Remote MCP made it accessible to anyone
