# Slide 11: The Adoption Reality

## Type
Landscape / state-of-the-world slide

## Content
- **Title**: "So Where Are We?"

- Three approaches and their adoption:

  | Approach | Who's doing it | Status |
  |----------|---------------|--------|
  | CLI / Bash | OpenClaw, various agents | ✅ Adopted |
  | Tool Search | Claude Code, Cursor | ✅ Adopted |
  | Code Mode | ...nobody? | ❌ Missing |

- "We thought people would adopt code mode. And they have... to some extent."
- But nobody has gone all-in on code mode for MCP

- **The blocker**: executing someone else's untrusted code is scary

## Layout
- Title top
- Table or three cards showing adoption status
- Callout box at bottom highlighting the blocker

## Key point
- The idea is proven but adoption is blocked by a trust/security problem
- Transition to: why untrusted code execution is hard
